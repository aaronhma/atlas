# Simulator
## THE COMPUTING MODULE ON ATLAS SIMULATOR!

## Getting Started
1. Go to root directory.
```bash
cd $PATH_TO_ATLAS/simulation # replace $PATH_TO_ATLAS with the place of your atlas
```

2. Run this command:
```bash
python3 -m pip install pyqt5 pudb
```

3. Then, start the simulator.
```bash
python3 main.py
```
